## ALLOWED\_FAN\_MODES \_CHANGED

Changes the value of the variable `FAN_MODES_LIST`.


### Name

`ALLOWED_FAN_MODES_CHANGED ()`


| Parameter | Type  | Description                                |
| --------- | ----- | ------------------------------------------ |
| `Modes`   | LIST. | A comma delimited list of available modes. |


### Returns

`None`
