## HAS\_HEAT\_PUMP

Boolean to enable/disable `has_heat_pump`. If the device supports this feature, default is false.


### Name

`HAS_HEAT_PUMP ()`


| Parameter | Description | Description                                |
| --------- | ----------- | ------------------------------------------ |
|           | BOOL        | True/False. enable/disable `has_heat_pump` |


### Returns

`None`
