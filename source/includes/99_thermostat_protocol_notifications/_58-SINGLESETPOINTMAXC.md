## SINGLE\_SETPOINT\_MAX\_C

Maximum Setpoint. Will do the same thing as the `setpoint_single_max` capability.

 
### Name

`SINGLE_SETPOINT_MAX_C ()` 


| Parameter | Type | Description       |
| --------- | ---- | ----------------- |
| SETPOINT  | INT  | 0-100 default 100 |

 
### Returns

`None`
