## CAN\_HUMIDIFY

Boolean to enable/disable `can_humidify` capability. If the device supports this feature, default is False.


### Name

`CAN_HUMIDIFY ()`


| Parameter | Type | Description                                          |
| --------- | ---- | ---------------------------------------------------- |
|           | BOOL | True/False. Enable/disable `can_humidify` capability |

### Returns

`None`
