## OUTDOOR\_TEMPERATURE\_RESOLUTION

What increment to use (ie .1, .5, 1, 3, 5, 10, etc), will do the same thing as the `outdoor_temperature_resolution` capability.


### Name

`OUTDOOR_TEMPERATURE_RESOLUTION ()` 


| Parameter | Description |
| --------- | ----------- |
| int       | Default 1.  |


### Returns

`None`
