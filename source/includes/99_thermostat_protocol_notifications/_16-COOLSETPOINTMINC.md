## COOL\_SETPOINT\_MIN\_C


### Name

`COOL_SETPOINT_ MIN_C ()`


| Parameter        | Type | Description                                                                               |
| ---------------- | ---- | ----------------------------------------------------------------------------------------- |
| Minimum Setpoint | INT  | 0-100 that will do the same thing as the [`setpoint_cool_min`][1] capability, default 100 |


### Returns

`None`
