## FAN\_MODE\_CHANGED

Notification that should be sent to the proxy when Fan mode has changed. Changes the variables: [`FAN_MODE`][1], [`V1 FANMODE,`][2] .


### Name

`FAN_MODE_CHANGED ()`


| Parameter | Type | Description                                       |
| --------- | ---- | ------------------------------------------------- |
| MODE      | STR  | One of the supported modes for example: Auto, On. |


### Returns

`None`
