## HEAT\_SETPOINT\_MAX\_F

Maximum Setpoint, int 0-100 that will do the same thing as the [`setpoint_heat_max`][1] capability.


### Name

`HEAT_SETPOINT_MAX_F ()`


| Parameter | Description |
| --------- | ----------- |
| int       | Default 100 |


### Returns

`None`
