## HVAC\_MODE\_CHANGED

Notification sent when the mode of the HVAC system has changed. Changes the variable `HVAC_MODE`.


### Name

`HVAC_MODE_CHANGED ()` 


| Parameter | Type | Description                                                             |
| --------- | ---- | ----------------------------------------------------------------------- |
| MODE      | STR  | One of the supported modes for example: Off, Heat,Cool, Auto, Emergency |


### Returns

`None`
