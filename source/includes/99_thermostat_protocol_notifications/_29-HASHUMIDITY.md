## HAS\_HUMIDITY

Boolean to enable/disable [`has_humidity`][1] capability. If the device supports this feature, default is false


### Name

`HAS_ HUMIDITY ()`


| Parameter | Type | Description                                                |
| --------- | ---- | ---------------------------------------------------------- |
|           | BOOL | True/False. enable/disable [`has_humidity`][2] capability. |


### Returns

`None`
