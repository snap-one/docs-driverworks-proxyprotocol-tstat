## CAN\_DEHUMIDIFY

Boolean to enable/disable `can_dehumidify` capability. If the device supports this feature, default is False.


### Name

`CAN_DEHUMIDIFY ()`


| Parameter | Type | Description                                              |
| --------- | ---- | -------------------------------------------------------- |
|           | BOOL | True/False.  Enable/disable `can_dehumidify` capability. |

### Returns

`None`
